SVN资源权限管理系统

SVN资源权限管理系统。一个安全、流畅、极简的远程SVN在线管理系统，解决了需要登录到远程服务器并通过命令行操作才能管理SVN服务器的烦恼，支持远程管理VisualSVN Server。

访问码云

技术选型

  1. 核心框架：Spring Boot。
  2. 安全框架：Apache Shiro。
  3. 模板引擎：Thymeleaf。
  4. 持久框架：MyBatis。
  5. 定时任务：Quartz。
  6. 更多……
